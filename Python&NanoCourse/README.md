# Estudos com Python - Nano Course FIAP

Este repositório contém os estudos realizados durante o Nano Course de Python oferecido pela FIAP, voltados para a obtenção de certificação.

## Conteúdo

- **Fundamentos de Python**: Estruturas de dados, condicionais, loops, funções.
- **Desenvolvimento de Scripts**: Automação de tarefas, manipulação de arquivos.
- **Geopy**: Trabalhos com geocodificação e localização geográfica utilizando a biblioteca Geopy.
- **FTP Cliente**: Implementação de um cliente FTP para transferências de arquivos.
- **Servidor TCP**: Criação de um servidor TCP para comunicação em rede.
- **Socket.py Cliente TCP**: Desenvolvimento de um cliente TCP usando o módulo `socket` para conexão com servidores.
